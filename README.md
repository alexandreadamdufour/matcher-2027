# Matcher 2027

MVP d'un comparateur de programmes pour l'élection présidentielle française de 2027, inspiré du Wahl-O-Mat allemand (Bundeszentrale für politische Bildung) et de Smartvote (Suisse).

## Démarche

Outil expérimental de civic-tech. L'objectif : explorer ce qu'un outil d'aide à la décision électorale à l'allemande donnerait dans le contexte français.

**Toutes les données de programmes sont fictives.** Aucun parti, aucun candidat réel n'est représenté. Ce projet ne collecte aucune donnée personnelle et ne stocke aucune réponse utilisateur.

## Méthodologie

Le matching suit le modèle Wahl-O-Mat : positions thématiques pondérées, calcul de proximité transparent (code source ouvert, algorithme auditable dans /src).

## Transparence

Projet open source précisément pour que la méthodologie soit vérifiable. Les contributions et critiques méthodologiques sont bienvenues via issues.
